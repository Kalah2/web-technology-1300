Thoughts on Web Accessibility

I think that web accessibility means that everyone including people with disabilities should be able to use and navigate anything on the internet. There are many reasons why the web should be accessible to everyone since it would make it easier for businesses to connect with a larger customer base which helps them get more loyal customers. Since the internet plays a big role in everyone's day-to-day life it would make it a lot easier for people to connect with eachother.

Final Project Revision and Reflection

Looking back at making my final project, I feel good about how it turned out, but I also see areas that could be improved. My main goal was to build a simple, clean website using only the HTML and CSS you taught us. While I am happy with the overall design, I realize I had too big of an idea for the website design so it took me longer to get started on the project than if I tried to do a smaller design and build on that I probably would have had a better design. If I were to redo the project, I would use some JavaScript so I could do more things design-wise a lot easier.
